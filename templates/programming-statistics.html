Programming & statistics projects

Throughout the years, I have done a lot of side projects involving many programming languages (to name a few, I have worked in Python, R, C, C++, C#, Javascript, Java, PHP, SQL, ...) and statistical analyses. But at the same time, I have this tendency to keep everything on my personal computer in unordered (and sometimes even unnamed) folders, and ultimately lose everything.

This page is an attempt to create a place where I feel compelled to gradually showcase some of these side projects. All codes can be found on my Github account.

(Click images to get to the code)

eyeScrollR

An open-source R package designed to map eye tracking gaze & fixation coordinates to unscrolled content images. Writing of a scientific article to present this package is in progress

Webscraping www.jeuxvideo.com

I wanted to try out BeautifulSoup (a Python webscraping library), so I used it to create a .csv file with all video game review grades from the most visited video game website in France

Disease spreading model

When Covid-19 vaccine disinformation started, I created a small video using a modified Conway's game of life in Python to show how non-perfect vaccines were still useful in a disease outbreak

Predictive ML business scenario

I did that one as a technical test in Python for a job interview. The idea was to predict bike renting around the San Francisco Bay Area in order to plan in advance when and where to move bikes between stations

Dice roll Android app

I did some consultancy as an Android programmer for a few months, yet somehow this is the only Android project I managed to recover. This was just a very simple dice rolling app to try out Kotlin for once instead of Java

Block Breaker

I wanted to try out Unity for a while, so I created this old-fashioned breaker game in C# from scratch. It's unfinished but playable (you can run it without building it in Android by downloading the "build" folder)

This website

It's not much, but this website runs on a simple Flask server
