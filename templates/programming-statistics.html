Programming & statistics projects

Over the years, I have worked on a variety of side projects involving various programming languages (to name a few, Python, R, C, C++, C#, Javascript, Java, PHP, SQL,...) and statistical analyses. At the same time, I have a habit of storing everything on my personal computer in unorganized (and sometimes unnamed) folders and eventually losing everything.

This page is an attempt to create a space where I can gradually showcase some of these side projects. All codes are available on my Github page.

(Click images to get to the code)

eyeScrollR

This is the first open-source R package designed to map eye tracking coordinates to unscrolled content images. A scientific article presenting this package is currently being written

Webscraping www.jeuxvideo.com

I wanted to try out BeautifulSoup (a Python webscraping library), so I used it to create a .csv file with all video game review grades from the most visited video game website in France

Disease spreading visualization

To combat the spread of Covid-19 vaccine misinformation during the pandemic, I made a short video in Python using a modified Conway's game of life to demonstrate why vaccines were important, and shared it on social media

Predictive ML business scenario

That was a Python technical test for a job interview. The idea was to use Machine Learning algorithms to forecast bike rental in the San Francisco Bay Area in order to plan when and where to move bikes between stations in advance

Dice roll Android app

For a few months, I worked as an Android consultant, but somehow this is the only Android project that I was able to salvage. This was just a very simple dice-rolling program to try Kotlin instead of Java

Block Breaker

As my first steps with Unity, I created this classic breaker game entirely from scratch using C#. It's playable but unfinished (you can run it without building it in Android by downloading the "build" folder)

This website

It's not much, but this website is powered by a simple Flask server which I wrote
